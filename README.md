# Dotfiles

Personal dotfiles for CachyOS Linux with fish shell, zsh shell, and Hyprland.

## Installation

```bash
cd ~/.dotfiles
./install.sh
```

## Structure

- `fish/` - Fish shell configuration
- `zsh/` - Zsh shell configuration
- `git/` - Git configuration 
- `starship/` - Starship prompt configuration
- `hypr/` - Hyprland configuration
- `scripts/` - Utility scripts
- `install.sh` - Installation script

## Features

### Fish Shell
- Custom aliases and functions
- Environment variables
- Auto-completion enhancements
- Custom prompt integration

### Zsh Shell
- Oh My Zsh integration
- Spaceship prompt theme
- Syntax highlighting and autosuggestions
- Same aliases and functions as fish

### Git
- Useful aliases
- Better diff and log formatting
- Sensible defaults

### Starship
- Custom prompt configuration
- Git integration
- Performance optimizations

## Backup

Before installation, your existing dotfiles are backed up to `~/.dotfiles_backup/`
